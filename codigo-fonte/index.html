Segurança e Praticidade

Realize suas doações de forma segura, basta realizar o cadastro e disponibilizar o melhor dia e horário para coleta. Os dados e informações dos usuários são disponíveis apenas para a instituição selecionada.

Agilidade

O cadastro para realizar as doações é simples e fácil de ser preenchido, uma vez realizado o cadastro o usuário está habilitado para realizar doações quando precisar.

Itens Doados

A sua doação é muito importante para nós, lembre-se de verificar os itens que são aceitos para doação por cada instituição.
